From a Python script to a full fledged application

A quick solution for creating rich applications with compelling interfaces for controlling your visualizations and data processing in plain Python. Trame enables you to skip the Web development part and just focus on your data and associated visualizations. Trame applications can be used locally like desktop applications, run in the cloud like web applications, or deployed on premise to access big and/or sensitive data. Trame comes fully featured by leveraging existing libraries and tools such as Vuetify, Altair, Vega, deck.gl, VTK, ParaView and more. VTK and ParaView are first-class citizens inside Trame, allowing you to be in full control of your 3D visualizations and data movement. Trame empowers any Python developer to create beautiful and interactive Web application in a very compact and intuitive way.
